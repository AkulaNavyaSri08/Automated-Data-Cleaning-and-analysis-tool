# Automated Data Cleaning Tool

## About
A Python-based tool that automatically cleans and analyzes datasets. It helps improve data quality by handling missing values, removing duplicates, fixing data types, and detecting outliers.

## Features
- Handles missing values
- Removes duplicate records
- Fixes incorrect data types
- Detects outliers
- Generates data quality reports
- Provides dataset insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
